CH3 Question.1

Given the following PDF: $${ f }_{ X }(x)=\begin{cases} A{ e }^{ -2x },\quad x\ge 0 \\ 0,\quad otherwise \end{cases}$$ where A is a constant.

(a) Find the value of A for ${ f }_{ X }(x)$ to be a valid PDF.
(b) Find its CDF ${ F }_{ X }(x)$.
(c) If ${ F }_{ X }(x)$ is the probability that a phone call lasts less than or equal to x minutes, find ${ F }_{ X }(5)-{ F }_{ X }(1)$ and state its meaning in words.

(a) 思考，然後自己動手嘗試

在(a)小題，他要找出A且題目所給的，是一個有效的PDF，於是我們回頭去想PDF有哪些性質：

CDF： ${ F }_{ X }(x)$、PDF：${ f }_{ X }(x)$

${ F }_{ X }(x)\xrightarrow [ \frac { d }{ dx } ]{ \xleftarrow { \int { } } } { f }_{ X }(x)$

${ { F }_{ X }(x)=\int _{ -\infty }^{ x }{ { f }_{ X }(u) } du }$

${ P(a\le X\le b)=\int _{ a }^{ b }{ { f }_{ X }(x) } dx }$

$\int _{ -\infty }^{ \infty }{ { f }_{ X }(x) } dx=1$

${ f }_{ X }(x)\ge 0$



現在讓我們來看來看 $\int _{ -\infty }^{ \infty }{ { f }_{ X }(x) } dx=1$ 這條公式，如這條公式所示，把PDF從負無窮大到正無窮大積分，得到1。

PDF是連續的random variables才有的，過去你學過random variable的意義X=1, X=2, X=...，X等於不同的值代表的是不同的events(事件)，(例如：你用X=1代表你準時到，X=2代表你遲到、X=3代表你缺席)。

random variable是一種簡寫的方式，讓你對事件的描述可以簡化。

積分是加總的概念，過去你曾在機率第三公理學過：events每個事件互相互斥，也在機率第二公理學過：樣本空間P(S)=1。

把樣本空間下所有的事件都加總起來，所得之和=1，就是$\int _{ -\infty }^{ \infty }{ { f }_{ X }(x) } dx=1$ 這條公式。

題目說的vaild probability，就是在說這條公式，也就是我們的機率第二與第三公理，所以我們透過這條公式就能得到A

現在你可以來嘗試看看了 ^_^



$\int _{ -\infty }^{ \infty }{ { f }_{ X }(x)dx=1 } $
$\Rightarrow \int _{ 0 }^{ \infty }{ A{ e }^{ -2x }dx=1 } $
$\Rightarrow \frac { A }{ -2 } { \left[ { e }^{ -2x } \right] }_{ 0 }^{ \infty }=\left( \frac { -A }{ 2 } \right) \left( 0-1 \right) =1$
$\Rightarrow A=2$

(b) 思考，然後自己動手嘗試

題目已經給我們PDF，所以我們可以透過${ F }_{ X }(x)\xrightarrow [ \frac { d }{ dx } ]{ \xleftarrow { \int { } } } { f }_{ X }(x)$來取得CDF

在此我們要注意的是積分的範圍，在上面我們提到CDF的公式：${ { F }_{ X }(x)=\int _{ -\infty }^{ x }{ { f }_{ X }(u) } du }$

依據題目所給的，${ f }_{ X }(x)$只有在$x\ge 0$時有值，所以我們積分的範圍，應該從CDF公式的$\int _{ -\infty }^{ x }{ } $，改成 $\int _{ 0 }^{ x }{ } $，說到這邊你應該有能力自己動手算了。



${ F }_{ X }(x)=\int _{ -\infty }^{ x }{ { f }_{ X }(t)dt } =\int _{ 0 }^{ x }{ 2{ e }^{ -2t }dt } $

$=-{ \left[ { e }^{ -2t } \right] }_{ 0 }^{ x }=1-{ e }^{ -2x }$

$\therefore { \quad F }_{ X }(x)=\begin{cases} 1-{ e }^{ -2x },\quad x\ge 0 \\ 0,\quad otherwise \end{cases}$

(c) 思考，然後自己動手嘗試

接下來我們先來讀一段前面的英文"If ${ F }_{ X }(x)$ is the probability that a phone call lasts less than or equal to x minutes."

這段話其實就是在說 ${ F }_{ X }(x)={ P }_{ X }(X\le x)$，CDF全名Cumilative distribution function，若你對discrite的CDF有印象的話，應該記得discrite CDF的圖畫起來是步階累加，最後會加總到1。

在離散的情況下，CDF他是累加events(即X=1, X=2, X=3,...)的機率(PMF)，所以CDF本身就是機率P，這很直觀，在連續的時候CDF就是去積分PDF所得的結果。

若給 ${ P }_{ X }(X\le x)$ 一個範圍，就是在這一個範圍的隨機變數發生的機率，${ F }_{ X }(x)={ P }_{ X }(a\le X\le b)$，也就是在這範圍內特定事件們發生的機率

在此的特定事件 ${ F }_{ X }(5)-{ F }_{ X }(1)$，是你打電話可能打大於1分鐘、1~2分鐘、1~5分鐘，剛好5分鐘但不會超過5分鐘

已經說了那麼多了，你有能力寫出來了 ^_^



${ F }_{ X }(5)-{ F }_{ X }(1)=\left( 1-{ e }^{ -10 } \right) -\left( 1-{ e }^{ -2 } \right) $

It is the probability that a phone call lasts less or equal to 5 minutes and greater than 1 minute.

CH3 Question.4

The lifetime of a certain kind of radio tubes, measured in hours, is assumed to have the following PDF: $${ f }_{ X }(x)=\begin{cases} 0,\quad x\le 100 \\ \frac { 100 }{ { x }^{ 2 } } ,\quad otherwise \end{cases}$$ What is the probability that exactly 2 out of 5 randomly selected radio tubes will have to be replaced within the first 150 hours of usage? (Assume lifetimes of different radio tubes are independent)

思考，然後自己動手嘗試

現在題目說，有一個特定的radio tubes真空管，它的lifetime生命週期，以小時來測量，已經知道lifetime的PDF，且已知lifetime之間互相獨立

現在我們來讀讀題目裡的這段英文"What is the probability that exactly 2 out of 5 randomly selected radio tubes will have to be replaced within the first 150 hours of usage?"

這段有點拗口，但你可以該嘗試閱讀它，如果你走工程，以後會看到更多這樣子的英文句子，現在我們來拆解這段句子。

"What is the probability that exactly 2 out of 5 randomly selected radio tubes" → 它要問機率，radio tubes隨機選5個，其中的2兩個

接著我們把"What is the probability that exactly 2 out of 5 randomly selected radio tubes"這句話簡化成 → (取radio tubes的事件)

所以原本的英文句子變成：(取radio tubes的事件) will have to be replaced within the first 150 hours of usage

這題開頭就說是在討論radio tube的lifetime，而lifetime是 measured in hours，不同長度的生命週期都是一個不同的事件，在這題中我們的random variable的對象是生命週期。

接著，我們回到完整的英文句子，把它們全部重讀一次：
What is the probability that exactly 2 out of 5 randomly selected radio tubes will have to be replaced within the first 150 hours of usage?

我們所要求的是機率

接著閱讀(取radio tubes的事件)，即"What is the probability that exactly 2 out of 5 randomly selected radio tubes"

我們隨機選五次中，有兩次是我要的

回到整個句子：(取radio tubes的事件) will have to be replaced within the first 150 hours of usage

我們隨機五次選時，剛好抽出兩次，這兩次要符合的條件是" within the first 150 hours of usage"

所以我們要求的是，隨機抽取五次，在這五次中有兩次的生命週期必須小於150小時的機率是多少

現在我們把小於150小時的機率看作P(成功)，大於150小時的機率看作P(失敗)=1-P(成功)

5次中要有2次成功，另外三次失敗

這樣子的機率可寫作 $\left( \begin{matrix} 5 \\ 2 \end{matrix} \right) { \left( P\left( sucess \right) \right) }^{ 2 }{ \left( P\left( fail \right) \right) }^{ 3 }$

P(失敗)=1-P(成功)，依題意所示，P(成功)是生命週期在150小時之內的機率，也就是生命週期小於等於150小時

題目已知PDF，PDF積分是CDF，CDF是特定範圍的機率，提示到這邊，你應該可以自己動筆試試看了



$\left( \begin{matrix} 5 \\ 2 \end{matrix} \right) { \left( P\left( sucess \right) \right) }^{ 2 }{ \left( P\left( fail \right) \right) }^{ 3 }$

我們需要有P(成功)才有辦法帶入上面那條式子求出所要的機率，依題意所示，P(成功)是生命週期小於等於150小時的機率

題目已知PDF，PDF積分是CDF，CDF是特定範圍的機率，成功的範圍是生命週期要小於等於150

P(成功)= ${ P }_{ X }(X\le 150)=\int _{ -\infty }^{ 150 }{ { f }_{ X }(x)dx=\int _{ 100 }^{ 150 }{ \frac { 100 }{ { x }^{ 2 } } dx } =\frac { 1 }{ 3 } } $

所以題目所求的機率 $=\left( \begin{matrix} 5 \\ 2 \end{matrix} \right) { \left( \frac { 1 }{ 3 } \right) }^{ 2 }{ \left( \frac { 2 }{ 3 } \right) }^{ 3 }=\frac { 8 }{ 243 } $

CH3 Question.6

Let X be a normal random variable with mean 0. In addition, we are given that $P[\left| X \right| \le 10]=0.1192.$. What is the standard deviation of X ? You may need to use the following table:

z 0.10 0.15 0.20 0.25 0.30
$\Phi(z)$ 0.5398 0.5596 0.5793 0.5987 0.6179

思考，然後自己動手嘗試

看到 $\Phi(z)$，我們先回來複習一下Gaussian distribution與Standard Normal Distribution

Gaussian distribution又稱之為Normal Distribution ，${ f }_{ X }(x)=\frac { 1 }{ \sqrt { 2\pi } \sigma } { e }^{ -\frac { { (x-\mu ) }^{ 2 } }{ 2{ \sigma }^{ 2 } } }$

通常Gaussian distribution我們會表示為 $X\sim Gaussian(\mu ,\sigma )$ 或 $X\sim N(\mu ,{ \sigma }^{ 2 })$，$\mu$ 代表expection 期望值，$ { \sigma }^{ 2 }$代表variance 變異數，$\sigma$代表standard deviation 標準差。

Standard Normal Distribution就是expection為0，variance為1，我們通常會把Standard Normal Distribution表示為 Z~N(0,1)。

由於Standard Normal Distribution的expection為0，variance為1，所以根據最上面Gaussian distribution的公式可得到

${ f }_{ Z }(z)=\frac { 1 }{ \sqrt { 2\pi } } { e }^{ -\frac { { z }^{ 2 } }{ 2 } }$

有了Standard Normal Distribution的PDF，我們對它積分，就能得到Standard Normal Distribution的CDF

Standard Normal Distribution的CDF $=\int _{ -\infty }^{ z }{ \frac { 1 }{ \sqrt { 2\pi } } { e }^{ -\frac { { z }^{ 2 } }{ 2 } }du } =\Phi (z)$，這個CDF的值積不出來，只能用查表的

現在我們來複習幾個Standard Normal Distribution的性質

對任何 $X\sim N(\mu ,{ \sigma }^{ 2 })$ 而言， $\frac { x-\mu }{ \sigma } \sim N(0,1)$

對任何 $X\sim N(\mu ,{ \sigma }^{ 2 })$ 而言， ${ F }_{ X }(x)=\Phi \left( \frac { x-\mu }{ \sigma } \right) $

${ \Phi }(-z)={ 1-\Phi }(z)$

在此我們特別推導一下上面三條中的第二條性質，待會會用到

${ F }_{ X }(x)=P(X\le x)$
$=P(X-\mu \le x-\mu )$
$=P(\frac { X-\mu }{ \sigma } \le \frac { x-\mu }{ \sigma } )$
$=\Phi (\frac { x-\mu }{ \sigma } )$

把$\frac { x-\mu }{ \sigma }$看作z，上面的式子就變為 $P(\frac { X-\mu }{ \sigma } \le z)=\Phi (z)$

現在我們可以回頭來看題目了，題目以知X是normal random variable 且mean=0，且已知$P[\left| X \right| \le 10]=0.1192.$，求X的standard deviation

我們先來看 $P[\left| X \right| \le 10]=0.1192$這條件，$P[\left| X \right| \le 10]=P[-10\le X\le 10]$，這是一個CDF

又由於題目所給的是normal ramdom variable，所以我們可用到上面三條性質中的第二條

說到這邊你可以嘗試動筆算算看了



${ F }_{ X }(x)=\Phi \left( \frac { x-\mu }{ \sigma } \right) $

$=P( X\le x)$

$\Rightarrow P[\left| X \right| \le 10]=P[10\le X\le 10]$

$=\Phi \left( \frac { 10 }{ \sigma } \right) -\Phi \left( \frac { -10 }{ \sigma } \right) $

透過剛剛說的最後一條性質

$=\Phi \left( \frac { 10 }{ \sigma } \right) -\left[ 1-\Phi \left( \frac { -10 }{ \sigma } \right) \right] =2\Phi \left( \frac { 10 }{ \sigma } \right) -1=0.1192$ by 題目條件

$\therefore \Phi \left( \frac { 10 }{ \sigma } \right) =0.5596$，查表得到 $\therefore \Phi \left( 0.15 \right) =0.5596$

$\therefore 0.15=\frac { 10 }{ \sigma } $

For the following joint PMF, prove that X and Y are not independent.
　 　 　 　 　 　 　 　 　 　
${ P }_{ XY }(x,y)$y=-2 y=0 y=2

思考，然後自己動手嘗試

題目要我們證明X、Y是獨立的

若兩個random variable是獨立的 $\Rightarrow { P }_{ XY }(x,y)={ P }_{ X }(x){ P }_{ Y }(y)$

若要使用上面這條公式我們需要 ${ P }_{ X }(x)$ 與 ${ P }_{ Y }(y)$

當我們已知joint PMF時，我們可以求得marginal PMF：${ P }_{ X }(x)$ 與 ${ P }_{ Y }(y)$
  1. ${ P }_{ X }(x)=\sum _{ y=-\infty }^{ \infty }{ { P }_{ XY }(x,y) } $
  2. ${ P }_{ Y }(y)=\sum _{ x=-\infty }^{ \infty }{ { P }_{ XY }(x,y) } $
現在題目已經給我們表了，那我們就隨便代一個joint PMF 像是 ${ P }_{ XY }(0,0)$，接著用marginal PMF與是否independent的公式驗證即可。

提示到這邊你應該有能力自己嘗試看看了 ^_^







${ P }_{ X }(0)=\frac { 1 }{ 8 } +\frac { 1 }{ 4 } +\frac { 1 }{ 8 } =\frac { 1 }{ 2 } $

$ { P }_{ Y }(0)=\frac { 1 }{ 4 } +\frac { 1 }{ 8 } =\frac { 3 }{ 8 }$

by 表格 ${ P }_{ XY }(0,0)=\frac { 1 }{ 4 } $，by independent的定義 ${ P }_{ XY }(0,0)={ { P }_{ X }(0)\cdot P }_{ Y }(0)=\frac { 3 }{ 8 } \times \frac { 1 }{ 2 } \neq \frac { 1 }{ 4 } $

所以 not independent，得證。

  2. ${ f }_{ Y }(y)=\int _{ -\infty }^{ \infty }{ { { f } }_{ XY }(x,y)dx } $，要求y就對x積分
提示到這邊你應該已經知道要怎麼寫了

兩兩相乘，與題目所給一樣，所以independent得證
